## Setup & config
1. Create an environment variable file .env

   ```
   
   PORT = PORT_VALUE
   user = "USERNAME_MYSQL"
   password = "PASSWORD_MYSQL"
   database = "DATBASE_NAME"
   SECRET_KEY = "JWT_SECRET_KEY"
   
   ```
2. Alter it with your real time variables
